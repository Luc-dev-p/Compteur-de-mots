Compteur de Mots (Lecteur de Fichiers)

Description
Ce programme Java lit le contenu d'un fichier texte (.txt) et calcule le nombre total de mots qu'il contient, en le parcourant ligne par ligne.

Concepts clés utilisés
Lecteur de fichiers : Utilisation de java.io.File et Scanner sur un fichier.
Gestion des Exceptions : Bloc try...catch pour gérer proprement l'erreur si le fichier est manquant (FileNotFoundException).
Manipulation de Strings : Découpage d'une chaîne de caractères avec la méthode split(" ").
Tableaux : Récupération de la taille d'un tableau avec .length.

 Comment lancer le projet ?
Clonez ce dépôt.
Ouvrez le projet dans IntelliJ.
Important : Assurez-vous qu'un fichier nommé monfichier.txt existe à la racine du projet et contient du texte.
Lancez la classe Main.
